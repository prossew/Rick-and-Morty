# Rick and Morty Gallery

This is a React application that displays a gallery of characters from the **Rick and Morty API**. The app is styled with **Styled Components** and allows users to filter characters by status, gender, species, and type.

---

## Features

- Display a gallery of characters with images and names  
- Filter characters using multiple criteria (status, gender, species, type)  
- Responsive layout for desktop and mobile  
- Popup window for character details (click background or Esc to close)
